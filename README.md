# Professional Personal Portfolio Website

**Live Demo:** [View Live Portfolio](https://portfolio-website-pink-six-83.vercel.app/)

## Project Overview
This project is a modern, fully responsive personal portfolio website designed and developed for the **Devixo Solutions Web Development Internship (Task 01)**. It serves to showcase my professional profile, technical skills, academic background, and project experience as a Full-Stack Developer and DevOps Engineer. The entire project is built strictly from scratch using core front-end technologies without relying on external frameworks like Bootstrap or Tailwind.

## Features
*   **Advanced Light/Dark Mode:** Seamlessly toggles between two drastically different premium UI themes.
*   **Vibrant Aurora Mesh (Light Theme):** A frosted glassmorphism aesthetic over a soft, animated, multi-colored mesh background.
*   **Deep Space Parallax (Dark Theme):** A pure CSS-based "Running Stars" background with 3D rotating glowing orbits.
*   **Fully Responsive Design:** Fluidly adapts to all screen sizes (mobile, tablet, desktop) using CSS Flexbox and Grid layouts.
*   **Interactive UI Elements:** Features smooth scrolling, hover animations, scaling cards, and custom-styled interactive buttons.
*   **Contact Form Interface:** A clean, user-friendly contact layout with custom placeholder instructions.

## Technologies Used
*   **HTML5:** For semantic, accessible, and structured web page layout.
*   **CSS3:** For advanced styling, CSS custom properties (variables), media queries, glassmorphism (`backdrop-filter`), and pure CSS animations (`@keyframes`).
*   **Vanilla JavaScript:** A minimal inline script strictly utilized for toggling the light/dark mode state.

## Setup Instructions & How to Run
Since this project is built entirely with static front-end technologies (HTML/CSS), no complex server setup or package installation is required to run it locally.

### Prerequisites
*   A modern web browser (Google Chrome, Firefox, Safari, or Edge).
*   A code editor like VS Code (optional, for viewing the code).

### Steps to Run Locally
1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/RanaSaad951/Portfolio-Website.git](https://github.com/RanaSaad951/Portfolio-Website.git)
    ```
2.  **Navigate to the Directory:**
    ```bash
    cd Portfolio-Website
    ```
3.  **Run the Project:**
    *   Simply double-click the `index.html` file to open it directly in your default web browser.
    *   *Alternatively*, if you are using VS Code, you can use the **Live Server** extension to open the project with hot-reloading capabilities.
